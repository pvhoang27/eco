# Shopee-like E-commerce Website

This project is a full-stack e-commerce web application inspired by Shopee.

## Features

- User authentication (register/login)
- Product listing and search
- Shopping cart
- Order management
- Admin dashboard
- Responsive, modern UI

## Tech Stack

- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
